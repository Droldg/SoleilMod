# Soleil

Soleil is a tModLoader mod that adds a custom boss, summon item, hostile projectiles, and the `/soleil` chat command.

## Development

Place this folder in `Documents/My Games/Terraria/tModLoader/ModSources/Soleil`, then build it through tModLoader:

1. Start tModLoader.
2. Open Workshop.
3. Open Develop Mods.
4. Run Build + Reload for Soleil.

The `/soleil` command gives the current player a Soleil Sigil, which summons the boss if another Soleil is not already active.
